용정시간표-Widget 설치 안내

1. 용정시간표-Widget-Setup.exe를 실행합니다.
2. Windows 보안 경고 또는 SmartScreen에서 "알 수 없는 게시자"가 표시될 수 있습니다.
3. 학교 내부에서 받은 설치 파일이 맞다면 아래 순서로 진행하세요.

SmartScreen 경고가 뜰 때:
- "추가 정보"를 누릅니다.
- "실행" 또는 "실행 허용"을 누릅니다.

알 수 없는 게시자 경고가 뜰 때:
- 파일 이름이 용정시간표-Widget-Setup.exe인지 확인합니다.
- 받은 위치가 신뢰 가능한 학교/관리자 전달 경로인지 확인합니다.
- 맞다면 설치를 계속 진행합니다.

주의:
- 이 경고는 프로그램에 문제가 있어서만 뜨는 것이 아니라, 코드 서명 인증서가 없는 앱에서 흔히 표시됩니다.
- 출처가 불분명한 곳에서 받은 파일은 실행하지 마세요.

설치 후:
- 시작 메뉴와 바탕화면에 용정시간표-Widget 바로가기가 생성됩니다.
- 위젯 안에서 "부팅 실행"을 켜면 Windows 시작 시 자동 실행됩니다.
- "라이트", "위에 고정", "투명도" 설정은 프로그램에 저장됩니다.
